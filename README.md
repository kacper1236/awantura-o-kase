# Awantura o kasę
Projekt jest w oparciu o teleturniej "Awantura o kasę". Napisany jest za pomocą Python i Django

# Uruchomienie aplikacji

## Wymagania
Python (testowany na wersji 3.10.X)
Django (najnowszy)

## Uruchomienie
W terminalu należy napisać
```
cd .\awantura_o_kase\
py manage.py runserver
```
